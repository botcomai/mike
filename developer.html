Quickstart Documentation

Make a POST request to our fulfillment endpoint to automatically purchase data from your Wallet Balance.

Endpoint

POST https://api.data4ghana.com/v1/fulfill

Headers

Authorization: Bearer YOUR_SECRET_API_KEY
Content-Type: application/json

Request Body (JSON)

{
  "merchant_id": "YOUR_MERCHANT_ID",
  "service": "data",
  "network": "MTN",
  "bundle_size_gb": 10,
  "recipient_phone": "0540000000"
}

Sample cURL Request

curl -X POST https://api.data4ghana.com/v1/fulfill \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{"merchant_id": "D4G-...", "network": "MTN", "bundle_size_gb": 10, "recipient_phone": "0541234567"}'
